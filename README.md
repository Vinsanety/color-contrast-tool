# Color Contrast Tool

Color Contrast Tool is a simple tool that helps designers and developers evaluate the accessibility of text and background color combinations. It calculates contrast ratios and checks compliance with [WCAG 2.1](https://www.w3.org/TR/WCAG21/) accessibility standards.

## Live Demo

Visit [Color Contrast](https://color-contrast-tool.web.app/)

## Features

- **Contrast Ratio Calculation**: Computes the contrast ratio between foreground and background colors.
- **WCAG Compliance**: Displays pass/fail results for AA and AAA standards (normal text, large text, and UI components).
- **Flexible Color Input**: Accepts HEX, RGB, and RGBA formats.
- **Live Preview**: Updates a demo box in real time to show how chosen colors look together.
- **Swap Colors**: Quickly switch foreground and background colors.
- **Input Synchronization**: Keeps text and color picker inputs aligned.

## Tech Stack

- **JavaScript (Vanilla)**
- **HTML5**
- **CSS3**

## Future Improvements
